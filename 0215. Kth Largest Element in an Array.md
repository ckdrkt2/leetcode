Bubble sort
```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        a = [None] * k
        for n in nums:
            for i in range(k):
                if a[i] is None: 
                    a[i] = n
                    break
                if a[i] < n:
                    a[i], n = n, a[i]
        return a[-1]
```

time complexity: O(kN)      
space complexity: O(k)

Tim sort

```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        return sorted(nums)[-k]
```
time complexity: O(NlogN)   
space complexity: O(N)
