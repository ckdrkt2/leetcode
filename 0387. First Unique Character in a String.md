class Solution:
    def firstUniqChar(self, s: str) -> int:
        a = 1e5
        for c in string.ascii_lowercase:
            l = s.find(c)
            r = s.rfind(c)
            if l == r != -1:
                a = min(a, l)
        return a if a < 1e5 else -1
```
1. a부터 z까지 탐색
2. 탐색할 문자를 왼쪽/오른쪽 방향으로 탐색하여 인덱스를 구함
3. 인덱스 값이 같고 -1이 아니면, 저장된 인덱스 값과 비교하여 작은 값을 저장
4. 탐색이 끝나면 저장된 인덱스 값이 정상 범위이면 리턴, 아니면 -1 리턴

space complexity: O(1)
- 최소 인덱스 값 공간만 사용

time complexity: O(N)
- 알파벳 수 x 문자열 길이 만큼 탐색
- 최대 O(26*N)이므로 O(N)

class Solution:
    def firstUniqChar(self, s: str) -> int:
        
        count = collections.Counter(s)
        
        for idx, ch in enumerate(s):
            if count[ch] == 1:
                return idx     
        return -1
```
해시 테이블 사용하여 탐색 횟수를 줄임

space complexity: O(1)
- 알파벳 수 만큼 공간 사용

time complexity: O(N)
- 문자열을 2번 탐색(테이블 1번, 인덱스 값 확인 1번)
- 최대 O(2*N)이므로 O(N)
