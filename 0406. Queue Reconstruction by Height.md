```python
class Solution:
    def reconstructQueue(self, people: List[List[int]]) -> List[List[int]]:
        people.sort(key = lambda x : (-x[0], x[1]))
        ans = []
        for p, k in people:
            ans.insert(k, [p, k])
        return ans
```

time complexity: O(NlogN)           
space complexity: O(N)
