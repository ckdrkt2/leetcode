```python
class Solution:
    def minOperations(self, n: int) -> int:
        return (n // 2) * (2 * (n % 2 + 1) + (n // 2 - 1) * 2) // 2
```

space compleixty: O(1)

time complexity: O(1)

문제에서 주어지는 리스트는 초항이 1이고 공차가 2인 등차수열이고 홀수 or 짝수에 따라서

리스트의 평균은 가운데 원소, 따라서 n // 2 개수만큼 평균보다 크고 모든 원소는 등차수열이기 때문에 평균과의 차이도 등차수열이 됨
결과적으로 n // 2개의 새로운 등차수열의 합이 리스트를 동등하게 만드는데 필요한 횟수

새로운 등차수열에서 홀수의 경우 초항이 2, 짝수의 경우 초항이 1이기 때문에 초항 값은 (1 + n % 2)가 된다.

결론은 수학적 해결방법으로 해결
